# AutoVista

The "rental-car-project" is a convenient and user-friendly application designed for a company specializing in providing car rental services in Ukraine.

## Overview

AutoVista is a comprehensive solution for car rental services, offering a seamless and intuitive user experience. Our application allows users to browse, search, and book rental cars with ease.

## Getting Started
To get started with this project, clone the repository and install the dependencies:

Copy code
```
git clone https://github.com/Daryna8/rental-car-project.git
cd rental-car-project
npm install
```
To run the application locally:

Copy code
```
npm run dev
```
